# Design Guidelines - Grupo Blanco Challenge

## Estilo visual

El juego debe verse como un simulador empresarial moderno.

Inspiración:
- Dashboard ejecutivo
- Sala de dirección
- Simulador estratégico
- Football Manager
- Juegos de gestión empresarial

El estilo debe ser:
- Profesional
- Corporativo
- Moderno
- Oscuro
- Claro en jerarquía visual

Evitar:
- Diseño de formulario
- Diseño académico aburrido
- UI genérica de SaaS
- Exceso de decoración sin función

## Jerarquía visual

La pantalla de juego debe priorizar:

1. Situación actual
2. Opciones de decisión
3. Indicadores de gestión
4. Consecuencia de la decisión
5. Progreso del año

## Ambientación Grupo Blanco

Grupo Blanco es una empresa argentina dedicada a:

- Importación de calzado
- Distribución mayorista
- Gestión de marcas internacionales
- Ecommerce
- Retail físico
- Franquicias
- Showrooms
- Logística
- Depósito
- Vendedores por zona

El juego debe usar términos como:

- Contenedor
- Importación
- Depósito Rosario
- Showroom Buenos Aires
- Canal mayorista
- Ecommerce
- Franquicia
- Retail
- Módulos cerrados
- Curva de talles
- Gerente de marca
- Comercio exterior
- Temporada alta
- Stock
- Clientes minoristas

## HomePage

La pantalla inicial debe explicar brevemente quién es Grupo Blanco.

Debe incluir:
- Qué hace la empresa
- Qué marcas gestiona
- Qué rol asume el jugador
- Qué indicadores deberá gestionar

Las marcas deben mostrarse como badges o etiquetas, no como logos oficiales.

Marcas:
- Skechers
- Havaianas
- Timberland
- Diadora
- Athix
- Comfortflex
- Ramarim
- Via Marte
- Kidy
- Boating
- Hey Dude

## GamePage

Debe sentirse como un “Tablero de Dirección General”.

Debe incluir:
- Header ejecutivo
- Nombre del jugador
- Rol: Director General
- Trimestre actual
- Escenario actual
- Timeline Q1/Q2/Q3/Q4
- Indicadores con barras de progreso
- Panel de situación actual
- Tres tarjetas de decisión
- Panel de consecuencia

## ResultPage

Debe mostrar:
- Calificación final
- Puntaje final
- Indicadores finales
- Fortaleza principal
- Debilidad principal
- Botón para jugar otra vez
- Botón para ver ranking
